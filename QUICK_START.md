## For Local Development

1. **Activate virtual environment**:
```bash
source venv/bin/activate
```

2. **Start the app**:
```bash
python app.py
```

3. **Access the app**:
- Open browser: `http://localhost:5001`
- Login with:
  - Username: `admin`
  - Password: `admin123`

4. **Change password** (IMPORTANT):
   - Go to Users page
   - Create a new admin user
   - Delete the old admin user
